HTML (Hyper Text Markup Language)

This is used to design WebPages like index.html, HomePage, AboutUsPage, etc.,. For every Web Application, the UI (User Interface) will be done using HTML Tim Berners-Lee created HTML in 1990 (Father of Web Environment)

Why do we need HTML?

HTML is the standard understandable code for every Web browser for UI Designing Every Web Application (developed using ASP.NET, PHP, Django, etc.,) render the output into client browser Html Tags.
